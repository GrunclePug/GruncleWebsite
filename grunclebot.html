GruncleBot

GruncleBot is a multi-purpose open source Discord bot written in Java.

Commands

  • Help | Displays the help menu
  • Serverinfo | Displays info on the current server
  • Userinfo {@user} | Displays info on user of choice
  • Info | Displays info on the bot
  • Ping | Check the ping between you and the bot
  • Invite | Gives you the invite to add the bot to a server

    • Fun Commands

  • Pat {@user} | Gives a user headpats
  • Hug {@user} | Gives a user hugs
  • Poke {@user} | Gives a user pokes
  • Slap {@user} | Gives a user slaps
  • Roll {# of sides} | Roll a dice with a given amount of sides

    • Music Commands

  • Play {youtube url} | Plays a song from youtube
  • Skip | Skips a song
  • Stop | Stops the player

    • Staff Commands

  • Purge {# of messages} | Purge x messages, where x is 2-100
  • Ban {@user} {reason} | Bans a user
  • Kick {@user} {reason} | Kicks a user
